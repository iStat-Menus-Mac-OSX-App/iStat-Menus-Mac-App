## About iStat Menus  

**iStat Menus for Mac** is a versatile monitoring app that displays real-time performance statistics for your Mac. Keep track of CPU load, GPU activity, memory usage, disk operations, network traffic, and battery status, all from the menu bar.  

The application is perfect for both casual and advanced users, offering a highly customizable interface. It supports Apple Silicon and Intel Macs, and is fully compatible with the latest macOS versions including Ventura and Sonoma. iStat Menus provides valuable insights to help maintain system health and optimize performance.
